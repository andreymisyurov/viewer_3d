демо-версия viewer 3d

Приложение написано на языке C++17 в графическом интерфейсе QT с использованием API OpenGL.
Для сборки используется Makefile с целями all, install, uninstall, clean, dvi, dist, tests.

Приложение реализовано с использованием архитектурного паттерна MVC (model - view - controller)
	контроллером выступает класс MainWindow, который находится в основной директории
	view - файл .ui лежащий в папке /view
	model - это файлы лежащие в папке /model

Использованы три паттерна проектирования:
	- Momento - сохраняет загруженный объект и его текущее положение
	- Singleton - реализован для парсера (по Майерсу)
	- Faced - реализован на контроллере

В данный момент работает левая половина интерфейса:
	- move, rotate, scale
	- загрузка 3д-модели из объектного файла
	- сохранение 3d модели с ее текущим положением в буфер
	- возможность покрутить модели при помощи мышки

В планах реализация:
